About

Coding Club adalah komunitas bagi siswa yang ingin belajar dan mengembangkan kemampuan di bidang teknologi. Kami fokus pada Web Design, Game Development, dan Python Programming. Dengan bimbingan yang menyenangkan, kami membantu siswa belajar coding dari dasar hingga mahir.

Our Divisions

Kami terbagi menjadi tiga divisi utama yang fokus pada keahlian berbeda.

Web Design

Divisi ini berfokus pada pembuatan tampilan website yang menarik dan responsif. Anggota divisi ini mempelajari HTML, CSS, JavaScript, serta framework seperti Bulma dan Bootstrap.

Game Development

Divisi ini mengembangkan berbagai game 2D dan 3D dengan menggunakan tools seperti Unity, Godot, atau Python. Fokus utamanya adalah kreativitas dan logika pemrograman.

Python Programming

Divisi Python berfokus pada pembelajaran logika pemrograman dan pengembangan aplikasi sederhana. Selain itu, anggota juga belajar otomasi dan analisis data dengan Python.
